# AIO Tools – All-in-One Chrome Extension

AIO Tools is a powerful and lightweight Chrome extension that combines multiple productivity tools into one clean interface. Instead of installing multiple extensions, you get everything in one place.

---

## Features

###  Tab Manager
- Save multiple tabs with one click
- Restore all saved tabs instantly
- Organize browsing sessions easily

###  Screenshot Tool
- Capture screenshots quickly
- Easy-to-use interface
- Fast and lightweight

###  Download Helper
- Simple file downloading support
- Clean and minimal workflow

### Right Click Enable
- Enable right-click on restricted websites
- Restore copy, paste, and inspect functionality

### More Features Coming Soon
- Full page screenshot
- Tab grouping
- Clipboard manager
- Dark mode
- And more...
